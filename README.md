# webpack-starter
A template for developing your angular frontend and express backend in one project using webpack-dev-server and nodemon.

## Usage
1. npm install
2. npm run dev
3. visit http://localhost:8888/

## Stack
* angular 1.6.4
* angular-material 1.1.4
* express 4.0.0

## Dev tools
* webpack components:
    * css-loader
    * style-loader
    * html-loader
    * babel-loader
    * html-webpack-plugin
* concurrently
* nodemon
